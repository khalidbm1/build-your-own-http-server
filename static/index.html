Build Your Own HTTP Server

A complete HTTP/1.1 server written from scratch in Go

v1.0

What is this?

This is a fully functional HTTP server built from the ground up using Go's standard library. No web frameworks — just raw TCP sockets, HTTP parsing, routing, and response building. Perfect for learning how the web works.

🔌 TCP Listener

Raw TCP connections without net/http framework

⚡ Concurrent

Handles 100+ simultaneous clients with goroutines

🛡️ Robust

Request validation, panic recovery, size limits

📍 Routing

URL routing and static file serving
